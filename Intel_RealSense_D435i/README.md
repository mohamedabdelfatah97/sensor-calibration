# Sensor Calibration

A growing collection of calibration pipelines for robotics sensors.
Each subfolder is self-contained with its own scripts, instructions, and results.

## Calibrated Sensors

| Sensor | Type | Method | Reprojection Error |
|---|---|---|---|
| Intel RealSense D455 | RGB Camera | OpenCV Checkerboard | 0.0226 px |
